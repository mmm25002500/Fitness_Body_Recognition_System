# ⚡ 快速開始

## 🚀 三步驟啟動

```bash
# 1️⃣ 安裝
bash install_mediapipe.sh

# 2️⃣ 啟動
bash run_app.sh

# 3️⃣ 開啟瀏覽器
# http://localhost:8501
```

## 📱 功能選擇

### 🤖 自動模式
AI 自動辨識 + 計數
- 上傳影片即可

### ✋ 手動模式
指定運動 + 計數
1. 選運動類型
2. 上傳影片

## 🎯 支援運動
1. 槓鈴彎舉
2. 錘式彎舉
3. 伏地挺身
4. 肩上推舉
5. 深蹲

## 💡 小技巧
- ✅ 全身入鏡
- ✅ 充足光線
- 🔄 重置：點側邊欄按鈕
- 🐛 除錯：勾選查看角度
